# DIGGS Team AI Handbook

A single-page, on-brand interactive guide for the DIGGS team on using Claude well: how to brief it, keep chats clean, reuse setups with Projects and Skills, pick a model, verify what matters, and stay safe with tools and connectors.

Self-contained static site. `index.html` plus `assets/` (Recta + Duplet Open webfonts, logos, one hero photo). No build step.

**Live:** https://adamdiggs.github.io/ai-handbook/

## Preview locally

```bash
python3 -m http.server 8731
```

Then open http://localhost:8731 (serve over http, not `file://`, so the fonts and image load).
